# bot-de-discord
Este bot de Discord permite a los usuarios subir imágenes para analizarlas automáticamente.

Cuando alguien envía una imagen con un comando, el bot la descarga y la procesa usando un modelo de inteligencia artificial entrenado en Keras Luego, el modelo predice a qué categoría pertenece la imagen y el bot muestra el resultado en el chat.

El bot también verifica que el archivo sea una imagen válida antes de analizarla.
